## Getting Started
1. [Join the forum](https://mcfam.forum/index.php?register/)
2. [Join a team](https://mcfam.forum/index.php?threads/fma_trading_comp.exe.29/#post-61)
3. Download your teams codebase.
### Unix (Linux/MacOS)
```shell
git clone git@github.com/umnfma/trade && cd trade/team/<your team number>
```
### Windows
#### Command Prompt/CMD
```cmd
git clone git@github.com:umnfma/trade.git & cd trade/team/<your team number>
```
#### Powershell
```powershell
git clone git@github.com:umnfma/trade.git; cd trade/team/<your team number>
```
4. Implement your trading strategy.
5. Push your implementation to the git repo.
```shell
git add .
git commit -m "Final Submission"
git push
```
